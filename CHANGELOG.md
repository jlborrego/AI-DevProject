# Changelog

## 1.0.0 - 2026-06-30

### Features
- Added Task Management API with create, read, update, and delete endpoints.
- Added a web-based TaskMaster dashboard UI at the app root for creating, editing, and deleting tasks.
- Enhanced the dashboard with Bootstrap styling, live search, status filters, task cards and modal editing.
- Added task validation for title, description, and status values.
- Added Playwright smoke and acceptance tests for the task API and dashboard entry point.
- Added documentation for running the project and API usage.
